Get Started

Deploy in Minutes

Choose your deployment method and have NexusOS running in under 5 minutes

🌐 Production Server

Deploy on your VPS or cloud server for enterprise use

  1. Download the pre-built Docker image
  2. Deploy to your server
  3. Access the web dashboard
# Download and deploy docker run -d \ -p 8080:8080 \ -p 3000:3000 \ -p 11434:11434 \ --name nexusos \ nexusos:full # Access your AI OS echo "Visit http://$(docker inspect nexusos | grep IPAddress | head -1):8080"
Requirements: Docker 20+, 4GB RAM, 10GB storage

💻 Local Development

Run NexusOS on your machine for development and testing

  1. Install Docker Desktop
  2. Pull the latest image
  3. Start developing immediately
# Get Docker Desktop first # https://www.docker.com/products/docker-desktop # Pull and run locally docker pull nexusos:full docker run -d \ -p 8080:8080 \ -p 3000:3000 \ -p 11434:11434 \ nexusos:full # Start building immediately echo "Visit http://localhost:8080"
Requirements: Docker Desktop, 4GB RAM, 10GB storage
