# Data Processing Agreement (DPA)

**Effective Date:** March 16, 2026

This Data Processing Agreement ("DPA") is entered into between NexusOS ("Processor") and the customer ("Controller") as a supplement to the Terms of Service.

## 1. Definitions

- **Personal Data:** Any information relating to an identified or identifiable person
- **Processing:** Any operation performed on personal data
- **Controller:** You (the customer)
- **Processor:** NexusOS

## 2. Scope and Purpose

This DPA applies to the processing of Personal Data by NexusOS in connection with the Service. The purpose of processing is to provide the AI agent platform functionality.

## 3. Categories of Data Processed

| Data Category | Purpose |
|--------------|---------|
| Account credentials | Authentication |
| User profile | Service delivery |
| Agent configurations | Agent functionality |
| Chat messages | Conversation processing |

## 4. Processing Activities

The Processor shall:
- Process Personal Data only on documented instructions from the Controller
- Ensure personnel are subject to confidentiality obligations
- Implement appropriate technical and organizational security measures
- Assist the Controller in responding to data subject requests
- Delete or return all Personal Data upon termination

## 5. Security Measures

The Processor implements:
- Encryption of data at rest and in transit
- Access controls and authentication
- Regular security testing
- Incident response procedures

## 7. Data Transfers

- **Self-hosted:** Data remains on Controller's infrastructure
- **Cloud services:** Data may be processed in the cloud provider's data centers
- Transfers comply with applicable data protection laws

## 8. Data Subject Rights

The Processor shall assist the Controller in fulfilling:
- Right to access
- Right to rectification
- Right to erasure ("right to be forgotten")
- Right to data portability
- Right to object

Response time: Within 30 days

## 9. Security Incident Notification

The Processor will notify the Controller within 72 hours of discovering any personal data breach.

## 10. Audits

Controller may audit the Processor's compliance with this DPA with reasonable notice.

## 11. Liability

Each party's liability is governed by the Terms of Service.

## 12. Termination

Upon termination, the Processor shall:
- Delete all Personal Data within 30 days
- Provide written confirmation of deletion
- Retain only what is legally required
