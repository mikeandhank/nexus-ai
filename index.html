Easy to Deploy

Get running in minutes with Docker

# Quick start with Docker
docker run -d -p 8080:8080 nexusos:latest

# Or install on your VPS
curl -sSL https://nexusos.cloud/install | bash

# Then open the web UI
http://localhost:8080

Powerful API

Everything in the UI, available via API

# Create an agent with personality
curl -X POST https://your-nexusos.com/api/agents \
  -H "X-Nexus-Key: sk-nexus-xxx" \
  -d '{"name": "assistant", "personality": "helpful, curious"}'

# Execute a task
curl -X POST https://your-nexusos.com/api/agents/agent-id/execute \
  -H "X-Nexus-Key: sk-nexus-xxx" \
  -d '{"task": "research market trends in AI"}'

# Create a swarm
curl -X POST https://your-nexusos.com/api/swarm/create \
  -H "X-Nexus-Key: sk-nexus-xxx" \
  -d '{"name": "research-team", "agents": [{"role": "researcher"}, {"role": "synthesizer"}]}'
