# Archives

**What belongs here:** Inactive items from Projects, Areas, or Resources.

- Completed projects (with post-mortem notes if valuable)
- Areas that are no longer active
- Superseded resources and outdated references
- Abandoned initiatives (note why they were abandoned)
- Historical records worth keeping

**What does NOT belong here:**
- Currently active projects (→ Projects)
- Ongoing responsibilities (→ Areas)
- Useful reference materials (→ Archives)

**Memory rule:** Archive is for retrieval, not active use. Review quarterly to consolidate or delete items that are no longer relevant.
