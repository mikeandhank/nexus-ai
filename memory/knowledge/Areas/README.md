# Areas

**What belongs here:** Ongoing responsibilities that require continuous maintenance. No deadline. No "done" state.

- Business functions (sales, operations, finance)
- Personal professional development
- Relationships that need ongoing nurturing
- Systems that need maintenance
- Recurring responsibilities

**What does NOT belong here:**
- Time-bound projects (→ Projects)
- Reference materials (→ Resources)
- Inactive areas (→ Archives)

**Memory rule:** Review area health regularly. If an area becomes inactive for 6+ months, move to Archives.
