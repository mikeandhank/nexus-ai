# Consumer Products & Marketplaces — Foundations

## Core Definition

**Consumer products** are physical goods sold to end consumers. **Marketplaces** are platforms connecting buyers and sellers without owning inventory. The core insight: **Distribution is the business** — the product is the reason to keep using it, but distribution determines whether you win.

---

## Marketplace Business Model

### Core Mechanics

| Element | Description |
|---------|-------------|
| **Supply** | Third-party sellers/vendors |
| **Demand** | Buyers/customers |
| **Platform** | Matches, enables transaction |
| **Revenue** | Commission, subscription, listing fees |

### Revenue Models

| Model | Description | Pros | Cons |
|-------|-------------|------|------|
| **Commission** | % per transaction | Aligns with success | Depends on GMV volume |
| **Subscription** | Recurring access fee | Predictable revenue | Churn risk |
| **Listing Fee** | Per item listed | Upfront revenue | Can deter sellers |
| **Advertising** | Promoted listings/sponsored products | High margin | Can hurt organic discovery |
| **Mixed** | Combination | Diversified | Complex |

### Key Metrics

- **GMV** (Gross Merchandise Value) — Total transaction volume
- **Take Rate** — Revenue / GMV (typically 5-20%)
- **LTV/CAC** — Lifetime value vs customer acquisition cost (target 3:1+)
- **Liquidity** — % of listings that sell (healthy: 20%+)
- **Match Rate** — Searches that result in purchase

## Network Effects

### The Virtuous Cycle (Flywheel)

1. **More buyers** → more sellers (better reach, more inventory)
2. **More sellers** → more buyers (more variety, better prices)
3. **More users** → better data → better matching → lower CAC

This creates a flywheel that compounds without proportional cost. But **network effects can reverse** — bad experiences drive users away faster than good ones attract them.

### Single vs. Double-Sided Network Effects

- **Single-sided** (e.g., social networks): Value increases with more users on one side
- **Double-sided** (e.g., marketplaces): Value increases when both sides grow simultaneously

## Key Challenges

### 1. Chicken-and-Egg Problem
- Can't attract sellers without buyers (nothing to sell)
- Can't attract buyers without sellers (nothing to buy)
- **Solution:** Seed one side with incentives, often losing money initially

### 2. Trust
- Buyers need confidence in sellers (quality, delivery, returns)
- Sellers need confidence in buyers (payment, returns)
- **Mechanisms:** Reviews, verification, dispute resolution, guarantees

### 3. Quality Control
- Platform doesn't own inventory
- Risks: Counterfeits, damaged goods, misrepresented items
- **Tradeoff:** Scale vs. quality — growth can overwhelm trust systems

### 4. Power Seller Problem
- Top 10% of sellers often drive 80%+ of volume
- If they leave, platform suffers
- **Risk:** Dependency on key suppliers

## Distribution Models

### Fulfillment Options

| Model | Description | Examples |
|-------|-------------|----------|
| **Dropshipping** | Seller ships directly to buyer | AliExpress, many Amazon 3P |
| **Platform Fulfillment** | Platform handles logistics | Amazon FBA, Shopify Fulfillment |
| **Hybrid** | Mix of both | eBay (some FBA, some direct) |
| **First-Party** | Platform owns inventory | Amazon Retail |

### The Fulfillment Arms Race

Amazon set the bar: Prime 2-day shipping is now table stakes. Competitors must match or differentiate on something else (selection, price, niche expertise).

## Consumer Products Dynamics

### How Winners Compete

| Dimension | What Matters |
|-----------|--------------|
| **Product** | Differentiation, quality, brand |
| **Price** | Margin management, economics |
| **Convenience** | Shipping speed, returns, UX |
| **Selection** | Breadth vs. depth of catalog |
| **Trust** | Brand, reviews, guarantees |

### The DTC Lesson (2020-2024)

Pure digital-first consumer brands struggled. Warby Parker survived by going omnichannel (145 → 230+ stores). Allbirds faltered because physical expansion proved harder than expected. **Channel strategy is product strategy.**
