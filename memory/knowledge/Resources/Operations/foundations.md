# Operations — Foundations

## Core Definition

Operations is the function of executing efficiently. It includes process design, resource management, quality control, and continuous improvement. Execution intelligence.

---

## Core Areas

| Area | Description |
|------|-------------|
| **Process** | How work gets done |
| **Supply Chain** | Getting inputs to outputs |
| **Quality** | Ensuring standards |
| **Capacity** | Managing throughput |
| **Improvement** | Getting better over time |

---

## Key Concepts

| Concept | Description |
|---------|-------------|
| **Efficiency** | Output per input |
| **Throughput** | Rate of production |
| **Cycle time** | Time per unit |
| **Bottleneck** | Limiting constraint |
| **Lean** | Eliminate waste |

---

## Frameworks

| Framework | Description |
|-----------|-------------|
| **Lean** | Minimize waste, maximize value |
| **Six Sigma** | Reduce variation |
| **Agile** | Iterate quickly |
| **Kanban** | Visualize workflow |
| **TOC** | Theory of Constraints |

---

## Core Principles

1. **Map your process**
2. **Find and fix bottlenecks**
3. **Quality is cheaper than rework**
4. **Automate after understanding**
5. **Continuous improvement**
