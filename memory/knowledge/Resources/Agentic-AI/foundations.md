# Agentic AI — Foundations

## Core Definition

Agentic AI refers to autonomous AI systems that can reason, plan, act independently, and collaborate to achieve goals. Unlike traditional AI (reactive, task-specific), agentic AI exhibits goal-directed behavior with minimal human intervention.

---

## Core Components

| Component | Description |
|-----------|-------------|
| **Memory** | Short-term and long-term context storage; past actions, conversations, outcomes |
| **Reasoning (Cognitive)** | LLM-powered goal definition, probabilistic decision-making, meta-learning |
| **Orchestration** | Coordinates agents via centralized, decentralized, or hybrid control flows |
| **Action/Execution** | Connects to APIs, tools, external systems for task execution |
| **Learning/Adaptive** | Refines through feedback, historical analysis, self-correction |
| **Collaboration** | Multi-agent coordination, human-AI interaction |

---

## Architecture Types

| Type | Description | Strengths |
|------|-------------|-----------|
| **Single-agent** | One agent handles focused problems | Simple, predictable |
| **Multi-agent** | Specialized agents collaborate | Scalable, flexible |
| **Vertical** | Lead agent delegates subtasks | Clear hierarchy, efficient |
| **Horizontal** | Peer agents work equally | Resilience, no single failure point |
| **Hybrid** | Mix of central/decentralized | Balances control and flexibility |

---

## How They Work

1. **Perception/Observation** — Agents observe environments via inputs, memory, tools
2. **Reasoning/Planning** — LLMs interpret goals, plan steps, decide actions dynamically
3. **Action/Execution** — Invoke tools/APIs, delegate to other agents, execute tasks
4. **Reflection/Learning** — Evaluate outcomes, update memory, adapt for future
5. **Orchestration Loop** — Workflow engines sequence interactions, handle errors

## Key Distinction

**Reactive AI** → Responds to inputs, task-specific, no memory
**Agentic AI** → Goal-directed, plans and executes, learns from context

The shift is from "answer questions" to "accomplish objectives."
