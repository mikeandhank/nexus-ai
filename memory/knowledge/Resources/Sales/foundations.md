# Sales — Foundations

## Core Definition

Sales is the process of converting prospects into customers. It includes prospecting, qualifying, presenting, handling objections, closing, and account management.

---

## Sales Models

| Model | Description |
|-------|-------------|
| **Outbound** | SDRs, cold outreach |
| **Inbound** | Leads come to you |
| **Account-Based** | Target specific accounts |
| **Self-Service** | Customer buys without reps |
| **Hybrid** | Mix of above |

---

## Sales Process

1. **Prospecting** — Find leads
2. **Qualification** — BANT, MEDDIC
3. **Discovery** — Understand needs
4. **Presentation** — Demo/solution
5. **Handling objections** — Address concerns
6. **Closing** — Get commitment
7. **Account management** — Expand, retain

---

## Key Metrics

| Metric | Definition |
|--------|------------|
| **CAC** | Customer Acquisition Cost |
| **Sales cycle** | Time from lead to close |
| **Win rate** | % of opportunities won |
| **ACV** | Annual Contract Value |
| **Quota attainment** | % hitting target |

---

## Core Principles

1. **Qualify hard** — Don't chase bad fits
2. **Listen more than talk**
3. **Value > features**
4. **Follow up is where deals are lost**
5. **Close for commitment every call**
