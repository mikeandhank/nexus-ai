# Resources

**What belongs here:** Reference materials, knowledge, and information that supports Projects and Areas.

- Notes and documentation
- External references and links
- Templates and frameworks
- Historical data and benchmarks
- Knowledge that has durable value

**What does NOT belong here:**
- Active project work (→ Projects)
- Ongoing areas (→ Areas)
- Outdated or superseded materials (→ Archives)

**Memory rule:** Resources should be easily searchable. If you find yourself consistently not referencing a resource, consider archiving it.
