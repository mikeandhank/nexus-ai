# Daily Morning Protocol

After 9AM research cron completes:

## What to Send Michael

Send via Telegram (message tool):
- **What I worked on** - Yesterday's progress, key findings
- **Plan for today** - 2-3 priorities I'm focusing on
- **Need from you** - Any decisions, approvals, or info I need

## Timing
- After 9AM research cron runs
- Before midday (give Michael time to review before afternoon)

## Example Format

```
Morning sync:

Yesterday: [what I accomplished]
- Key finding from research: [1 insight]
- Fixed: [bug fix or improvement]

Today: [my priorities]
1. [Priority 1]
2. [Priority 2]
3. [Priority 3]

Needs: [anything blocking or needing your input]
```
