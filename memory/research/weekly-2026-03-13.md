# Weekly Market Deep Dive — March 13, 2026

## AI Agent Funding Landscape

The agentic AI space is heating up fast. **Poolside** just closed a massive $500M Series B — the largest single agentic round. **Harvey** leads with $966M total (legal vertical), followed by **Sierra** at $635M (customer service). **Parloa** and **Lovable** each crossed $500M+ with recent rounds. Median funding across 100 tracked agent startups sits at $75M, with 48 of them raising in the last 12 months.

## SMB Automation Market

The software-defined automation market is valued at **$54.09B in 2026**, growing at 16% CAGR. SMEs represent ~45% of the market. Key barriers: 43% still prefer on-premise for security, and 37% cite high integration costs.

## Claude Code vs OpenAI Operator

Claude Code = local, interactive, developer-guided copilot (92% HumanEval). OpenAI Codex/Operator = cloud-based autonomous agent (77% SWE-Bench). Claude excels at complex reasoning; Operator excels at delegated async tasks. Cost: Claude 2-3x more expensive.

## Competitive Takeaways

- Vertical-specific agents (legal, customer service) are winning big checks
- SMB automation is massive but adoption stuck on integration complexity
- Developer tools split: interactive copilot vs autonomous agent — hybrid workflows emerging
